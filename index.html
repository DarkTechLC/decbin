Conversor decimal binário

Adicione valores em algum dos campos e clique em "Converter", o valor convertido será mostrado no campo que ficou vazio.

Processo de conversão de decimal para binário

Para realizar a conversão de um número decimal para binário, divida-o por 2, até obter quociente igual a 0, então, pegue os restos das divisões, começando pelo último. Dessa maneira, o número estará convertido para binário.

Processo de conversão de binário para decimal

Para converter um número de binário para decimal, pega-se cada bit do número, multiplicado pela potência de 2 elevado a posição dele, onde a contagem das posições inicia em 0 e da direita para a esquerda. Após obter o valor de cada posição, soma-se todos eles e o número está convertido para decimal.
